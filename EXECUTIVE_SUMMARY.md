## 🛠️ 技术栈

### 前端
```
Flutter 3.24.0 + Dart
├── Provider (状态管理)
├── Dio (HTTP客户端)
├── Hive (本地存储)
└── Material Design 3 (UI)
```

### 后端
```
FastAPI 0.120.4 + Python 3.10+
├── SQLAlchemy (ORM)
├── SQLite/PostgreSQL (数据库)
├── Redis (缓存)
├── Celery (任务队列)
└── Uvicorn (服务器)
```

### 基础设施
```
ComfyUI (AI引擎)
Docker (容器化)
Git (版本控制)
```
