## 📊 技术栈

### 后端
- **框架**: FastAPI 0.120.4
- **数据库**: SQLite (开发) / PostgreSQL (生产)
- **ORM**: SQLAlchemy 2.0.44
- **认证**: JWT + PBKDF2
- **异步**: FastAPI BackgroundTasks

### 前端
- **框架**: Flutter 3.x
- **状态管理**: Provider + Riverpod
- **HTTP客户端**: Dio 5.3.0
- **本地存储**: Flutter Secure Storage

### 测试
- **后端**: pytest 8.3.5 + pytest-asyncio
- **前端**: flutter_test + mockito
- **集成**: FastAPI TestClient
