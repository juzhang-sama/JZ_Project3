## 🔧 技术栈

### 前端
- **框架**: Flutter 3.24.0
- **语言**: Dart
- **状态管理**: Provider + Riverpod
- **HTTP客户端**: Dio 5.3.0
- **本地存储**: Hive 2.2.0
- **UI**: Material Design 3

### 后端
- **框架**: FastAPI 0.120.4
- **语言**: Python 3.10+
- **ORM**: SQLAlchemy
- **数据库**: SQLite (开发) / PostgreSQL (生产)
- **缓存**: Redis
- **任务队列**: Celery
- **服务器**: Uvicorn 0.33.0

### 基础设施
- **AI引擎**: ComfyUI
- **容器**: Docker
- **版本控制**: Git
