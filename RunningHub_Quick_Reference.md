## 🛠️ 技术栈速查

### 前端技术栈
```
框架：Nuxt.js 3.x
语言：Vue 3 + TypeScript
构建：Vite
国际化：i18n
状态管理：Pinia
HTTP：Axios
```

### 后端技术栈
```
语言：Python 3.x
核心框架：ComfyUI（开源节点工作流引擎）
API框架：FastAPI（推测）
任务队列：Celery / RQ
```

### 数据库技术栈
```
主数据库：PostgreSQL
缓存：Redis
文件存储：对象存储（OSS/COS）
```

### 基础设施
```
计算：GPU集群（NVIDIA/AMD）
容器：Docker
编排：Kubernetes
加速：CDN
```
